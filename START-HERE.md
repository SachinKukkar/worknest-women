## 📋 Deployment Checklist

Follow these steps in order:

1. ✅ **Push to GitHub** (5 min)
   - Create GitHub repo
   - Push your code

2. ✅ **Setup MongoDB Atlas** (5 min)
   - Create free cluster
   - Get connection string

3. ✅ **Deploy Backend (Render)** (5 min)
   - Connect GitHub repo
   - Add environment variables
   - Deploy

4. ✅ **Seed Database** (2 min)
   - Run seed command in Render Shell

5. ✅ **Deploy Frontend (Vercel)** (3 min)
   - Connect GitHub repo
   - Add API URL
   - Deploy

6. ✅ **Update CORS** (2 min)
   - Update backend with frontend URL
   - Redeploy

**Total Time**: ~20 minutes  
**Total Cost**: $0/month (100% FREE)
